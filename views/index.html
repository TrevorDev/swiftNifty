Order and pay on the go!

The future of mobile payments is here.

WHAT WE DO?

Allow you to buy/sell goods securly with your mobile phone or browser.

Why do we need this?

I have always hated waiting in lines to pay for something with cash or credit. We should all have the option to order and pay at the same time without being restricted by a human behind a counter. Swift Nifty solves this problem by allowing people to place an order safely and simply with a mobile phone.

Open Data!

Swift Nifty has utilized open data from multiple resourses. Using the open data from Google maps api Swift Nifty can easily display store location/directions. Additionally yelp provides an open data reviews api which makes it easy for users to see what's good.

How is this secure?

All payments are processed through the Stripe API using the https protocal. Stripe creates a payment token that only Swift Nifty can use to make a transaction. This means Swift Nifty doesn't handle any credit card information keeping your private data private.
